# React Practice Website
This project is a small frontend web application built with **React** to practice basic React concepts and component-based architecture.
The interface is based on a template website and was recreated as a learning exercise.

## Technologies Used
* React
* JavaScript
* HTML
* CSS
* Bootstrap

## Concepts Practiced
* Component-based structure
* React Router (BrowserRouter)
* useEffect hook
* Basic React project structure

## Live Demo
https://react-practice-web.netlify.app

## Purpose
The purpose of this project was to practice building a frontend application using React and understand how routing and components work in a React project.
